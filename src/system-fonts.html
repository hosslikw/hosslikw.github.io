System Fonts

included here are different fonts that come pre-installed to the different operating systems. The goal of this project was to give developers a visual reference of the system fonts they have downloaded to their computers. If you're making a fallback font list in your CSS you might use this resource to better understand your defaults and they styles you've chosen. If you are seeing a large amount of repetition it is because each font in this list is set to default to either 'Indie Flower' or 'Courier New' if your operating system doesn't have the stated font type.

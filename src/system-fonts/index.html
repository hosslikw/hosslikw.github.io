System Fonts

The fonts below come installed on at least one of the predominant computer operating systems; Mac, Windows, and Linux. If the listed font-family is native to your sytem or you have otherwise downloaded a font, its style will render here in bright text. Perhaps you are writing a font fallback list in CSS or you are curious about changing the default font of your browser. Each font is set to render as "regular/medium" to be the standard size of your user interface which is 16pt in If you're making a fallback font list in your CSS you might use this resource to better understand your defaults and they styles you've chosen. If you are seeing a large amount of repetition it is because each font in this list is set to default to either 'Indie Flower' or 'Courier New' if your operating system doesn't have the stated font type.
